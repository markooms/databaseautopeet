| `TRELLO_LABEL_ADDED` | Label toegevoegd | webhook | `trello_label`, `trello_user` |
| `TRELLO_ARCHIVED` | Kaart gearchiveerd | webhook | `trello_user` |

## Webhook Data: Wat kunnen we nog meer opslaan?

### Huidige situatie

We slaan nu op:
- `trello_label` = label **naam** (bijv. "mark ooms")

### Probleem

Label namen kunnen wijzigen. Beter is om de **label ID** op te slaan.

### Aanbevolen aanpassingen

**1. Nieuwe kolom in vacature_events:**
```sql
ALTER TABLE vacature_events ADD COLUMN trello_label_id VARCHAR(50);
```

**2. Nieuwe dimensie-tabel voor labels:**
```sql
CREATE TABLE trello_labels (
    trello_label_id   VARCHAR(50) PRIMARY KEY,
    naam              VARCHAR(100),
    kleur             VARCHAR(20),
    medewerker_id     UUID,  -- FK naar toekomstige medewerkers tabel
    is_actief         BOOLEAN DEFAULT TRUE,
    created_at        TIMESTAMPTZ DEFAULT NOW(),
    updated_at        TIMESTAMPTZ DEFAULT NOW()
);
```

**3. Toekomstige medewerkers tabel:**
```sql
CREATE TABLE medewerkers (
    medewerker_id     UUID PRIMARY KEY,
    naam              VARCHAR(200) NOT NULL,
    email             VARCHAR(200),
    trello_label_id   VARCHAR(50) REFERENCES trello_labels(trello_label_id),
    profiel_url       VARCHAR(500),  -- Link naar interne profielen site
    is_actief         BOOLEAN DEFAULT TRUE,
    created_at        TIMESTAMPTZ DEFAULT NOW()
);
```
